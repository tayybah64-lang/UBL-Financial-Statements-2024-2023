# UBL Financial Statements Analysis 2024 vs 2023

**Complete Excel-based reconstruction and analysis** of United Bank Limited (UBL) financial statements for the year 2024 compared to 2023.

## 📊 Project Overview
- Full reconstruction of Statement of Financial Position, Profit & Loss, Comprehensive Income, Changes in Equity, and Cash Flow Statement.
- Professional **Dashboard** with key highlights, growth analysis, and ratios.
- Multiple charts for better visualization.
- Detailed supporting notes.

## 🛠 Key Features
- Fully linked Excel model with clean professional formatting
- Growth Comparison Table (2024 vs 2023)
- Key Ratios (ROE, ROA, NIM, Cost-to-Income, ADR)
- Visual Charts: Asset Composition, Growth Trends, Profit & Income Breakdown
- Key Observations section

## Key Performance Highlights
- **Total Assets** grew **36.7%** to Rs 8.07 trillion
- **Net Advances** increased by **65.5%**
- **Profit After Taxation** rose **36.3%** to Rs 75.16 billion
- **Total Income** surged **39.0%**
